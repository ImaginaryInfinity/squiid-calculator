# Greater Than
`gt`

The `gt` command will test if the left number is greater than the right number. Returns 1 if true, 0 if false.

----

### Function Arguments
```plaintext
gt(left_operand, right_operand)
```

This is equivalent to `left_operand > right_operand`

----

### Algebraic Example
```plaintext
gt(3, 4)
```

### RPN Example
```plaintext
3
4
gt
```

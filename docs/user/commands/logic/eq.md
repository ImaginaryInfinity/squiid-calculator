# Equals
`eq`

The `eq` command will test if two numbers are equal. Returns 1 if true, 0 if false.

----

### Function Arguments
```plaintext
eq(left_operand, right_operand)
```

This is equivalent to `left_operand == right_operand`

----

### Algebraic Example
```plaintext
eq(3, 4)
```

### RPN Example
```plaintext
3
4
eq
```

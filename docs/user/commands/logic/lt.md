# Less Than
`lt`

The `lt` command will test if the left number is less than the right number. Returns 1 if true, 0 if false.

----

### Function Arguments
```plaintext
lt(left_operand, right_operand)
```

This is equivalent to `left_operand < right_operand`

----

### Algebraic Example
```plaintext
lt(3, 4)
```

### RPN Example
```plaintext
3
4
lt
```
